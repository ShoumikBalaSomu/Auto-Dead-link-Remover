Auto Dead-link Remover

The ultimate background app for Android TV that automatically cleans your IPTV playlists. Never encounter a dead link again.

Features:

  • 📺 Runs seamlessly in the background on any Android TV.
  • 🔄 Periodically checks your IPTV playlist and removes dead channels.
  • 🌐 Embeds a local HTTP server (`localhost:8080`) to stream clean playlists.
  • 🚀 Auto-starts on boot for a hassle-free experience.
